What are the differences between relative, absolute and fixed positioning?


Relative, absolute and fixed positioning are ways we can manipulate elements using the CSS position property. Before we define each, it is important to know that if no positioning value is defined, the element will always default to position-static. Meaning when you define specifically that an element be tagged position static, the positioning outcome will be the same as the default document flow. Remember that term, well come back to it later.

    position: relative

  • position: relative is the command we use to affect the positioning of an element relative to itself; meaning it will move however many pixels, percentages or ems relative to what its position-static value was.


    • We see here that the blue box has been positioned 20px away from its previous top and left fixed positions. This is what is meant by relative to itself.


    Furthermore the space created by moving the element, will remain unfilled by any surrounding elements. Relative positioning can also be used for parent elements to define the relationship of nested child elements with absolute positioning

      position: absolute

  • This one is powerful but a little dangerous. position: absolute allows you to position exactly where you want. But there are a few rules to remember. If you set absolute positioning values of height and width your element will position itself relative to the document itself disregarding any element its nested within.

      • position: fixed

  • This one is like absolute but even more subborn! Position fixed will break document flow again, meaning the space the element once occupied is now fillable by other elements. But unlike absolute positioning, fixed elements do not conform to the relative positioning of parent elements, nor are they effected by scrolling at all. So say for instance you want a banner add on your page to be positioned at exactly 20px from the top and left of your page, no matter what elements it overlaps, and you want it to remain there when the user scrolls, you would want position: fixed on that ad.
